search

Searches the specified table for the keywords against the fields. Returns the fields specified under the keys parameter. Has two types, 'like' and 'range', where like is a basic search against fields and range is a search between two points (like dates) Can be passed a custom string that will execute any custom where clauses you may want.

public

Parameters
keywordsarray or string an array or comma delimited string holding the keywords to search for. eg. array('1', 'two') or '1, two'
fieldsarray or string an array or comma delimited string holding the fields to search against. eg. array('field1', 'field2') or 'field1, field2'
tablestring the name of the table you want to search
keystring a comma delimited string that specifies which fields to return. eg. 'field1, field2'
typestring which type of search to perform. valid types are 'like' and 'range'
customstring A custom where clause to add the the query. Must be a valid MySQL comparison. eg. 'field1 is NULL'
Returns
Returns the array of search results if successful, false if not.
